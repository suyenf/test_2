
# Prueba Tecnica

Prueba de desarrollo para cargo de backend


## Ejecutando en local

Clonar el proyecto

```bash
  git clone https://github.com/suyenf/test_2.git prueba
```

entrar al directorio

```bash
  cd prueba
```

configurar el archivo .env

```bash
  cp .env.example .env
  php artisan key:generate
```

se deben establecer credenciales de base de datos dentro de este archivo

```env
# archivo .env
...
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laravel
DB_USERNAME=root
DB_PASSWORD=
...
```
instalar dependencias

```bash
  composer install 
  npm install
```

compilar front-end

```bash
  npm run dev
```

inicar el server

```bash
  php artisan serve
```
